 # Tasky

A simple and clean Todo application built with **React** and **Supabase**. This project demonstrates basic CRUD operations with a modern frontend and a backend database.

## Features

- Add tasks
- View tasks
- Delete tasks
- Responsive UI

## Tech Stack

- React
- Vite
- Supabase
- JavaScript
- CSS

## Upcoming Features

- Edit tasks
- Mark tasks as completed
- Search tasks
- Dark mode
- Authentication
